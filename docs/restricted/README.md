# Restricted documents (private)

This folder is reserved for documentation that is **not safe to publish publicly**.

Policy: `docs/policy/POL-DOCS-PUBLICATION.md`.

Rules:

- Put safety-sensitive operational and security details here.
- Do not link to this folder from public documentation.
- Do not publish this folder to the public website.
- It is OK to version this folder in the repo for internal sharing and backup, but manage visibility and access controls at the repo level and keep live secrets out of the repository.
