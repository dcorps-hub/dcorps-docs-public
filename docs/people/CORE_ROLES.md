> Scope: Core roles needed to execute Phase 0 and Phase 1.

---

## Phase 0 roles

- Protocol lead (Cosmos SDK, state machine design).
- Go/Cosmos engineer (core modules and protobufs).
- Data/indexer engineer (schema, ingestion, reporting).
- DevOps/SRE (environments, CI/CD, monitoring).
- Security lead (threat model, audits, bug bounty).
- Frontend engineer (explorer and console UX).

---

## Phase 1 roles

- Governance and policy coordinator.
- Protocol release manager.
- Ecosystem support and integration lead.
